  • ETIQUETAS

    • Titulos y textos

      En HTML los títulos se definen con las etiquetas h1> hasta h6>, siendo h1> el más importante y h6> el menos jerárquico. Para el texto normal se usa p> para párrafos, br> para saltos de línea y hr> para separar secciones con una línea horizontal.

      Style

      En CSS las propiedades más comunes para dar estilo al texto y a los elementos son color{} para cambiar el color de la letra y background-color{} para el color de fondo, font-size{} para el tamaño de la tipografía, font-family{} para elegir la fuente. También se usan text-align{} para alinear el texto (izquierda, derecha, centro, justificado). Para el diseño de cajas son muy comunes margin{} para el espacio exterior, padding{} para el espacio interior, border para los bordes y width junto con height para tamaño. Otras propiedades frecuentes son display para definir cómo se comporta el elemento (bloque, inline, flex, grid), position para ubicar elementos en la página. Finalmente, se usan mucho background-image{}, background-size{} y background-repeat{} para manejar imágenes de fondo.

      Multimedia

      En HTML la etiqueta principal para imágenes es img>, que se usa para insertar una imagen dentro de la página mediante el atributo src y un alt como texto alternativo. Para videos se utiliza video>, que permite incrustar archivos de video; dentro de video> se coloca source> para indicar la ruta del archivo en distintos formatos. Para mapas de imagen se usa la etiqueta map>

      Estructura y contenido

      Son etiquetas HTML de estructura y contenido, que sirven para organizar y dar significado a lo que aparece en una página web. Algunas, como div>, son genéricas y solo agrupan elementos para darles estilo o funcionalidad; otras son semánticas, como section>, nav> y article>, que no solo agrupan contenido, sino que le dan un sentido específico: section> para bloques temáticos, nav> para menús o navegación, y article> para piezas de contenido independiente. En conjunto, todas forman parte del “esqueleto” que le da orden y lógica a un documento HTML, permitiendo que tanto los navegadores como los usuarios (e incluso los buscadores) entiendan mejor la información.

  • IA

    • ConferUACJ

      Una plataforma digital que permite a los estudiantes registrarse, recibir confirmación y asegurar su ingreso a conferencias de manera organizada y sin saturación de cupos.

      Funcionalidades principales

      Registro rápido: Formulario con nombre, matrícula, correo y carrera.

      Visualización de eventos: Lista de conferencias disponibles con cupo restante.

      Confirmación de asistencia: Generación de un código QR único para ingresar.

      Control de aforo: Sistema que evita sobrepasar la capacidad del lugar.

      Panel administrativo: Permite a los organizadores ver quién está registrado, cupo restante y enviar recordatorios.

      Flujo de usuario

      1. El estudiante abre la app → ve la lista de conferencias → selecciona una.
      2. Llena su información y confirma registro.
      3. Recibe un código QR de ingreso.
      4. En la entrada, el código QR es escaneado → registro confirmado → asistencia contabilizada.
